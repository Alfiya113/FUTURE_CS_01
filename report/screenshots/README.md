# Screenshots & Evidence

This folder contains screenshots captured during the passive security analysis
performed for Cyber Security Task 1 under the Future Interns Internship Program.

## Types of Screenshots Included
- Nmap scan results (open ports and exposed services)
- OWASP ZAP passive scan alerts
- Security headers inspection
- Browser Developer Tools analysis
- Any other relevant non-intrusive findings

## Purpose
Screenshots serve as visual evidence to support the findings documented in the
vulnerability assessment report.

## Note
All screenshots were captured using read-only and ethical techniques.
No active exploitation or intrusive testing was performed.
